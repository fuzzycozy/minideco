block 요소

특징) 줄내리기 효과, 가로 사용할 수 있는 전체 가로영역 설정, h와 p태그의 자손태그로 block태그 사용 불가

h1~h6 : 타이틀 태그 (숫자가 커질 수록 타이틀 비중 적음)
div : box형태의 콘텐츠를 그룹
article : box형태의 콘텐츠를 그룹 (조건: 자식으로 h태그를 반드시 작성해야함)
p : 문서의 단락이 되는 콘텐츠 그룹
ul,li : 순서가 없는 목록 or 리스트 콘텐츠에 사용

가장 큰 타이틀 (문서에 한 번만 사용)

두번째로 큰 타이틀

세번째로 큰 타이틀

네번째로 큰 타이틀

다섯번째로 큰 타이틀
여섯번째로 큰 타이틀

텍스트 또는 이미지 태그를 만들 때 사용

  • 리스트 형식의 콘텐츠를 만들 때 사용
  • 여백과 불릿을 기본값으로 설정하고 있다
  • 콘텐츠 유형(메뉴, 카드형 콘텐츠, 게시판 리스트...)

inline 요소

*특징*
한줄내리기 없음
콘텐츠의 크기만큼 가로영역 설정
inline요소 안에 block요소 사용 금지
style명령 : 사이즈와 여백 사용 못함

span : 인라인 요소들 그룹
img : 이미지 URL 호출
a : 다른 문서로 이동 (링크)
strong,b : 텍스트 bold
i, em : 텍스트 italic
mark : 텍스트 배경색 생성
텍스트, 이미지와 같은 inline 콘텐츠를 그룹 네이버로 이동
  • strong 강조(bold)
  • b(bold)
  • em 강조(italic)
  • i(italic)
  • mark 배경색으로 강조
